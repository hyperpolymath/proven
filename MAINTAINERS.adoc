=== Core Library (`src/Proven/`)

All 104 verified Idris2 modules with dependent type proofs.

* **Primary:** Jonathan D.A. Jewell

=== Zig FFI Bridge (`ffi/zig/`)

Pure C ABI bridge layer connecting Idris2 RefC output to language bindings.

* **Primary:** Jonathan D.A. Jewell

=== Language Bindings (`bindings/`)

120+ language binding targets. Thin wrappers only — no reimplemented logic.

* **Primary:** Jonathan D.A. Jewell

* **Primary:** Jonathan D.A. Jewell
