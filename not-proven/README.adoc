= not-proven — Modules That Do Not Compile
:toc:

== Purpose

This directory contains Idris2 modules that were removed from the main
`src/Proven/` tree because they **do not compile** with Idris2 0.8.0
and therefore cannot be formally verified.

== Why They Were Moved

Each module has specific compilation errors documented below. Common patterns:

* **Undefined names**: Functions referenced but never defined (batch-generated stubs)
* **Type mismatches**: String vs List, Bool vs Type, wrong Nat operations
* **Missing instances**: Eq, Foldable, Show not derived for custom types
* **Keyword conflicts**: `total`, `prefix`, `record` used as variable names
* **Covering/totality**: Recursive functions not provably total

== Status

These modules are **NOT DELETED** — they contain useful type signatures and
proof obligations that document what SHOULD be proven. They serve as a
specification for future work.

== How to Restore

To work on fixing a module:

1. Copy it back: `cp not-proven/SafeFoo.idr src/Proven/SafeFoo.idr`
2. Fix the compilation errors
3. Verify: `idris2 --find-ipkg --check src/Proven/SafeFoo.idr`
4. If it passes, commit the restoration
