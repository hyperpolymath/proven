= proven: Quickstart Guide
:toc: macro
:icons: font
:source-highlighter: rouge

Formally verified safety library. All computation happens in Idris2 with
dependent types. Zig bridges to C ABI. Language bindings call through FFI only.

toc::[]

== Prerequisites

[cols="1,1,2"]
|===
| Tool | Version | Install

| Idris2
| >= 0.7.0
| `pack install idris2` (via https://github.com/stefan-hoeck/idris2-pack[pack])

| Zig
| >= 0.13.0
| `asdf install zig latest` or https://ziglang.org/download/

| just
| >= 1.0
| `cargo install just` or https://just.systems

| pack
| latest
| https://github.com/stefan-hoeck/idris2-pack#installation

| Guile Scheme
| >= 3.0
| `dnf install guile30` / `apt install guile-3.0`
|===

Optional (for bindings work):

* Rust >= 1.75 (`asdf install rust nightly`)
* Deno >= 2.0 (for ReScript bindings)

== Clone and build

[source,bash]
----
cd ~/Documents/hyperpolymath-repos
git clone https://github.com/hyperpolymath/proven.git
cd proven

# Full build: Idris2 -> RefC -> Zig FFI -> libproven.so
just build
----

Build takes 2-5 minutes. On success:

----
[proven] Step 1: Compiling Idris2 to C via RefC backend...
[proven] Step 2: Building Zig FFI library...
[proven] Zig FFI build complete.
[proven] Output:
-rwxr-xr-x 1 you you 245760 ... libproven.so
[proven] Full build complete.
----

== Typecheck the full library (258 modules)

[source,bash]
----
just typecheck
----

This verifies all 258 modules compile and pass totality checking.

== Use a binding

The shared library at `ffi/zig/zig-out/lib/libproven.so` can be loaded from
any language with C FFI support. See `bindings/` for 120+ language targets.

Example (Rust, from `bindings/rust/`):

[source,bash]
----
cd bindings/rust
cargo test
----

== Troubleshooting

`idris2 not found`::
  Install via pack: https://github.com/stefan-hoeck/idris2-pack

`build/refc/ not found`::
  Run `just build-refc` before `just build-ffi`. Or run `just build` which does both.

`zig: error: ... asdf shim`::
  Ensure `.tool-versions` has `zig 0.13.0` and run `asdf reshim zig`.

`libidris2_support not found`::
  Run `idris2 --prefix` and verify the support libraries exist at that path.
